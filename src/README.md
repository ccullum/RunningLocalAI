# 💻 Source Code Directory

This directory contains the core Python scripts for interacting with the local LM Studio instance. 

The code is structured progressively into "Lessons" to demonstrate the iterative development of the AI assistant, starting from a basic connection and moving toward advanced streaming and analysis features.

### Directory Structure:
* **`Lesson-01/`**: Core connection mechanics and basic API testing.
* **`Lesson-02/`**: Advanced response handling, including streaming outputs and data analysis.
